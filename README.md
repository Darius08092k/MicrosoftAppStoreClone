# Microsoft App Store Clone

A WPF desktop application that replicates the Microsoft Store user interface and functionality. This project demonstrates modern desktop application development using Windows Presentation Foundation (WPF) with a Metro design style.

## Features

- **App Store Interface**: Clean, modern UI mimicking the Microsoft Store
- **App Browsing**: View featured apps, popular apps, free apps, and games
- **App Details**: Comprehensive app information pages with multiple tabs:
  - Overview
  - System Requirements
  - Reviews
  - Related Apps
- **Downloads & Updates**: Dedicated page for managing app downloads and updates
- **Top Apps Section**: Browse top-performing applications
- **Theme Customization**: Accent color and theme selection
- **Smooth Animations**: Fade-in animations for enhanced UX
- **Responsive Design**: Adaptive UI that adjusts to window size

## Prerequisites

- **Windows 10 or later**
- **.NET Framework 4.7.2** or higher
- **Visual Studio 2019+** (for development)

## Dependencies

- **MahApps.Metro** - Metro design framework for WPF
- **MiscUtil** - Utility library for random number generation
